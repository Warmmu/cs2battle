# 更新日志

## [1.0.0] - 2025-11-06

### 新增功能

#### 核心功能
- ✅ 玩家注册系统
  - 支持昵称注册
  - 可选 Steam ID 绑定
  - 初始 ELO 1000

- ✅ 房间系统
  - 自动创建/加入房间
  - 支持最多 10 人
  - 实时状态同步
  - 准备机制

- ✅ 匹配系统
  - ELO 平衡算法
  - 自动分队
  - 最小化队伍 ELO 差

- ✅ BP 系统
  - 地图 Ban/Pick 流程
  - A Ban → B Ban → A Pick
  - 支持 7 张官方地图

- ✅ 比赛系统
  - 比分录入
  - KDA 统计
  - MVP 评选
  - 自动 ELO 计算

- ✅ 数据统计
  - 个人历史战绩
  - 全服排行榜
  - ELO 历史曲线
  - 胜率、KD 统计

#### 技术特性
- 微信云开发架构
- 云函数后端
- 云数据库存储
- 响应式 UI 设计
