# 地图图片文件夹

## 📍 说明

此文件夹用于存放 CS2 地图的预览图片，用于 BP 页面展示。

## 📦 需要的图片

请在此文件夹中放置以下 7 张地图图片：

| 文件名 | 地图名称 |
|--------|---------|
| `inferno.jpg` | 炼狱小镇 |
| `mirage.jpg` | 荒漠迷城 |
| `dust2.jpg` | 炙热沙城II |
| `nuke.jpg` | 核子危机 |
| `overpass.jpg` | 死亡游乐园 |
| `ancient.jpg` | 远古遗迹 |
| `anubis.jpg` | 死神殿堂 |

## 🎨 图片规格

### 建议规格
- **尺寸**：600 × 400 像素（3:2 比例）
- **格式**：JPG 或 PNG
- **大小**：每张 < 200KB
- **质量**：清晰可辨

### 命名规则
- 全部小写
- 使用英文地图名
- 不要使用空格

## 🔍 图片来源

### 方法 1：官方素材
从 CS2 官方网站或 Steam 社区下载地图预览图

### 方法 2：游戏截图
在游戏中截取地图全景图，裁剪为合适尺寸

### 方法 3：第三方资源
从 CS 相关社区网站下载（注意版权）

### 方法 4：占位图
如果暂时没有图片，可以使用纯色占位图：
- 背景色：`#667eea`
- 文字：地图名称
- 尺寸：600 × 400 px

## ⚠️ 注意事项

1. **版权**：确保使用的图片有使用权限
2. **性能**：图片过大会影响加载速度
3. **一致性**：所有图片保持相同尺寸比例
4. **备份**：建议保留原图备份

## 🛠️ 图片优化工具

- **在线工具**：TinyPNG、Squoosh
- **本地工具**：Photoshop、GIMP
- **批量处理**：ImageMagick

## 📝 示例

正确的文件结构：
```
miniprogram/images/maps/
├── inferno.jpg      (150KB)
├── mirage.jpg       (145KB)
├── dust2.jpg        (160KB)
├── nuke.jpg         (155KB)
├── overpass.jpg     (148KB)
├── ancient.jpg      (152KB)
└── anubis.jpg       (158KB)
```

## 🚀 快速开始

如果你需要快速测试，可以暂时使用占位图：

1. 创建 600×400 的纯色图片
2. 添加地图名称文字
3. 保存为对应文件名
4. 后续替换为真实地图图片

---

**重要**：此文件夹中必须包含所有 7 张地图图片，否则 BP 页面会显示异常！
